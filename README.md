## Installed Plugins

Plugin list:

- **[folke/lazy.nvim](https://github.com/folke/lazy.nvim)**: A modern plugin manager for Neovim, bootstrapped in the config.
- **[navarasu/onedark.nvim](https://github.com/navarasu/onedark.nvim)**: A dark colorscheme for Neovim.
- **[MeanderingProgrammer/render-markdown.nvim](https://github.com/MeanderingProgrammer/render-markdown.nvim)**: Renders Markdown elements directly inside Neovim buffers.
- **[nvim-treesitter/nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter)**: Advanced syntax highlighting and parsing.
- **[nvim-mini/mini.nvim](https://github.com/nvim-mini/mini.nvim)**: Mini.nvim suite dependency used by `render-markdown.nvim`.
- **[folke/which-key.nvim](https://github.com/folke/which-key.nvim)**: Shows available keybindings in a popup.
- **[obsidian-nvim/obsidian.nvim](https://github.com/obsidian-nvim/obsidian.nvim)**: Obsidian vault integration for Neovim.
- **[neovim/nvim-lspconfig](https://github.com/neovim/nvim-lspconfig)**: Quickstart configurations for Neovim LSP.
- **[nvim-mini/mini.files](https://github.com/nvim-mini/mini.files)**: File explorer from the mini.nvim plugin collection.
- **[folke/lazydev.nvim](https://github.com/folke/lazydev.nvim)**: Better Lua development setup for Neovim config/plugin development.
- **[sphamba/smear-cursor.nvim](https://github.com/sphamba/smear-cursor.nvim)**: Animated smear-style cursor movement for Neovim.
- **[saghen/blink.cmp](https://github.com/saghen/blink.cmp)**: A performant completion engine for Neovim.
  - **[rafamadriz/friendly-snippets](https://github.com/rafamadriz/friendly-snippets)**: Snippet collection installed as a dependency for `blink.cmp`.
